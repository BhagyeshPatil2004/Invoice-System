# Invoice System

## Project Info

A modern, efficient invoice management system built with React and TypeScript.

## Getting Started

To run this project locally:

1.  **Clone the repository**
    ```sh
    git clone https://github.com/BhagyeshPatil2004/Invoice-System.git
    ```

2.  **Navigate to the project directory**
    ```sh
    cd Invoice
    ```

3.  **Install dependencies**
    ```sh
    npm install
    ```

4.  **Start the development server**
    ```sh
    npm run dev
    ```

## Technologies Used

This project is built with:

-   Vite
-   TypeScript
-   React
-   shadcn-ui
-   Tailwind CSS

## Deployment

To deploy this project, build it using:

```sh
npm run build
```

Then host the `dist` folder on your preferred static site hosting provider (e.g., Vercel, Netlify).
